# ⚡ 30秒更换 PWA 图标 - 超快指南

## 🎯 三步搞定

### 1️⃣ 生成图标（本地）

```bash
npm run dev
```

浏览器打开：`http://localhost:5173/generate-icons.html`

点击三个下载按钮 👇

---

### 2️⃣ 保存文件

把下载的三个文件拖到项目的 `/public` 目录：

```
public/
├── icon-192.png          ← 拖进来
├── icon-512.png          ← 拖进来
└── apple-touch-icon.png  ← 拖进来
```

---

### 3️⃣ 手机测试

#### **方法A：同一WiFi（快速）**

```bash
# 终端会显示：
➜ Network: http://192.168.1.100:5173
           ↑ 手机浏览器访问这个地址
```

#### **方法B：iOS 需要 HTTPS**

```bash
# 安装 ngrok
brew install ngrok

# 启动隧道
ngrok http 5173

# 访问显示的 https 地址
```

---

## 📱 添加到主屏幕

**iOS:** 分享按钮 📤 → "添加到主屏幕"  
**Android:** 菜单 ⋮ → "安装应用"

✅ **图标立即生效！**

---

## 🔍 验证

访问这些地址确认图标存在：
- `http://localhost:5173/icon-192.png` ✅
- `http://localhost:5173/icon-512.png` ✅
- `http://localhost:5173/apple-touch-icon.png` ✅

## 🐛 遇到问题？

**图标不显示？**
```bash
# 1. 清除浏览器缓存
# 2. 重启开发服务器
npm run dev

# 3. 使用隐身模式测试
```

**手机访问不了？**
```bash
# 确保同一 WiFi
# 或者用手机热点

# Windows 防火墙：允许 Node.js
# macOS 防火墙：允许传入连接
```

## 📊 对比：本地 vs 部署

| 方式 | 优点 | 缺点 |
|------|------|------|
| **本地测试** | ✅ 即时修改<br>✅ 无需等待<br>✅ 免费 | ⚠️ 需要同一网络<br>⚠️ iOS 需要 HTTPS |
| **部署 Vercel** | ✅ 永久 HTTPS<br>✅ 任意设备访问 | ⏰ 需要等待构建 |
